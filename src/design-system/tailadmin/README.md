# TailAdmin Design System

Camada visual interna adaptada do TailAdmin para dashboards administrativos e area do participante.

Regras:

- Nao importar componentes diretamente de `free-nextjs-admin-dashboard-main`.
- Nao aplicar estes tokens na Home publica sem decisao explicita.
- Manter a identidade publica ATLAS separada da identidade interna dos dashboards.
- Usar azul TailAdmin/institucional como cor principal interna.
- Usar dourado apenas como acento pontual quando necessario.
